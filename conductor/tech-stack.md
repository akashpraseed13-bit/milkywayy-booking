# Technology Stack - Milkywayy Booking

## Frontend
- **Framework:** Next.js (App Router)
- **Library:** React 19
- **Styling:** Tailwind CSS v4, PostCSS
- **Components:** Radix UI (Shadcn UI components)
- **Icons:** Lucide React

## Backend & Database
- **API:** Next.js Route Handlers
- **ORM:** Sequelize
- **Database:** PostgreSQL
- **Object Storage:** AWS SDK (S3)

## Authentication & Security
- **Authentication:** Custom JWT-based (using `jose` and `jsonwebtoken`)
- **Password Hashing:** Bcrypt
- **Validation:** Zod

## Payments & Integrations
- **Payments:** Stripe
- **Forms:** React Hook Form with Zod resolvers
- **Social Media:** React Social Media Embed

## Development Tools
- **Linting & Formatting:** Biome
- **Database CLI:** Sequelize-cli
